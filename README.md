Blinkit Sales Dashboard (DVA Project)

This project is a Data Visualization and Analysis project built using Google Sheets for data cleaning and Looker Studio for dashboard creation.
The objective of this project is to analyze Blinkit sales data and generate meaningful business insights using interactive visualizations.

Project Overview

Blinkit is a quick commerce grocery delivery platform operating across multiple outlet locations. This project analyzes its sales dataset to understand sales performance, outlet distribution, product trends, and revenue patterns.
Workflow followed in this project:
Raw Dataset → Data Cleaning in Google Sheets → Dashboard in Looker Studio

Tools Used

Google Sheets for data cleaning and preprocessing
Looker Studio for dashboard visualization
CSV dataset

Data Cleaning Process (Google Sheets)

The following steps were performed:
Handling missing values
Standardizing inconsistent category names
Removing duplicate entries
Formatting numeric and rating columns
Creating calculated fields
